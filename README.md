# MyStore

## Overview

MyStore is a responsive e-commerce web application built with React and Vite. Users can browse products, search, filter, sort, view product details, and manage a shopping cart.

## Features

- Home page
- Product listing
- Product details
- Search
- Category filtering
- Product sorting
- Shopping cart
- Local storage
- Responsive design
- Toast notifications

## Technologies

- React
- Vite
- React Router
- Tailwind CSS
- Axios
- React Toastify
- Fake Store API

## Installation

```bash
npm install
npm run dev
```
